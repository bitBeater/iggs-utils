# ecma-utils

[See Documentation](https://bitbeater.github.io/ecma-utils/)

Simple, lightweight, dependenciesless, TypeScript generic helper library.

---
Compatible with:
- Node.js
- Deno
- Browser

## Installation

npm
```bash
npm i @bitBeater/ecma-utils
```

deno
```bash
deno install npm:@bitBeater/ecma-utils
```
